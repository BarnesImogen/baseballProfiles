## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Usage

### Viewing Profiles
1. Navigate to the home page
2. Search for players or browse all profiles
3. Click on a player card to view their full profile

### Creating a New Profile
1. Click "Create New Player Profile" on the home page
2. Fill in the required information
3. Add stats, experience, and other details
4. Click "Create Profile" to save

### Editing a Profile
1. Navigate to a player's profile
2. Click "Edit Profile" in the navigation
3. Make your changes
4. Click "Save Changes"
